## Techstack

 - **NextJs** - Full stack framework
 - **Tailwind** - styling
 - **ShadCN** - UI Components
 - **Clerk** - Authentication
 - **Neon** - Postgre DB
 - **Prisma** - ORM
 - **Upload thing** - Images
 - **Vercel** - Deploy application
